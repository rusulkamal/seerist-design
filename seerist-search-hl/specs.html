Search Highlight
When a keyword search is active, all titles dim to regular weight. Matched words stay bold so they stand out without colour or decoration.
Behaviour
Idle
Parliament blocks sanctions bill amid protest
All titles at bold weight. No treatment applied.
Search active
Parliament blocks sanctions bill amid protest
Titles shift to regular. Matched words stay bold.
Tokens
Default
font-weight: 700
Search active
font-weight: 400 transition: 150ms ease
Matched word
font-weight: 700 same color
Scope
Title only — date, tags, source unchanged
Matching is case-insensitive. Unquoted terms match independently — protest sanctions highlights each word. Quoted phrases match exactly — "protest sanctions" only highlights the full sequence.
Animation
Search starts
All titles shift from 700 → 400 over 150ms. Matched words wrap in a <mark> and stay at 700, creating contrast without any colour change.
Search clears
Titles transition back to 700 over 150ms. Without this, clearing the input snaps all visible titles simultaneously — jarring on a long list.
