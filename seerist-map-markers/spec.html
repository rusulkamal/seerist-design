01

Visual Language

Each visual property of a marker carries exactly one meaning. Nothing is decorative.

Shape → Content type
Circle = Event. Rounded square = Analysis or News & Social. Filled circle = Asset.
Border → Event status
Solid = Verified. Dashed = Unverified. Dashed + clock = Future. Dashed + spark = Breaking.
Colour → Severity
Red = high. Orange = medium. Grey = low.
Icon → Category
The icon shows the type category — terrorism, maritime, unrest, etc.

03

Event Markers

Incidents on the ground, produced by Seerist. Circle shape, 34 × 34 px.

Statuses

Verified — Solid border
Confirmed by Seerist. Most common event status.
Unverified — Dashed border
Reported but not yet independently confirmed.
Future — Dashed + clock
Scheduled or anticipated event. Clock badge bottom-right.
Breaking — Dashed + spark
Happening right now. Spark badge top-right. Animated pulse ring.

Severity

LevelColourHex
HighRed#dc2626
MediumOrange#ea580c
LowGrey#78716c

Breaking pulse animation

Breaking events emit a slow pulse ring to signal urgency without being distracting.

ScaleExpands from 0.9× to 1.8× the marker size
OpacityFades from 0.45 → 0 over the cycle
Duration2.6 seconds per cycle
StaggerEach breaking marker offsets by 0.28 s so they don't all pulse in sync

04

News & Social Markers

Third-party reporting — news articles, wire feeds, and social media posts. Not independently verified by Seerist.

ShapeRounded square (squircle) — immediately distinguishable from circular event markers
Size34 × 34 px — same as events, equal visual weight on the map
BorderSlate #94a3b8 — third-party, lower trust
IconMatches the event category (bullhorn, anchor, etc.) so users can compare directly
Visible fromZoom level 2 and above
05

Analysis Markers

In-depth written assessments produced by Seerist analysts. Green border signals Seerist-authored, high trust.

ShapeRounded square — same as News & Social, both are intel content rather than ground events
Size34 × 34 px
BorderGreen #10b981 — Seerist-authored, high trust
IconDocument / file-lines — represents a written report. Icon colour matches the green border.
Visible fromZoom level 2 and above
06

Asset Markers

The organisation's own tracked locations — offices, field teams, embassies, vessels, facilities. Completely separate from events and intel.

ShapeCircle with solid colour fill — instantly distinct from all other markers (no white background)
Size32 × 32 px
ColourEach asset type has its own colour set by the organisation
ClusteringNever clustered — always visible at all zoom levels
Layer toggleToggled independently via the Assets layer in the Layers panel
07

Cluster Markers

When multiple events are geographically close at the current zoom level, they merge into a cluster bubble to keep the map readable.

AppearanceCircle with a count number. Neutral slate border and dark text — no severity colouring.
Size34 px (≤7 events) · 40 px (8–15) · 46 px (16+)
PositionPlaced at the geographic centroid of all member events
To expandZoom in — clusters break apart automatically at higher zoom levels

Zoom thresholds

Zoom levelBehaviour
Below 2.5Maximum clustering — world view
2.5 – 5Moderate — regional view
5 – 7Loose — country view
7+No clustering — all individual markers visible
08

Same-Location Stack

When two or more events share the exact same coordinates, they are shown as a stacked group with a double-border circle and a count.

AppearanceCircle with inner border and outer ring — double-border effect makes it visually distinct from single events
CountNumber inside reflects all events at that location
ClickOpens a list in the popover — select any event to see its full detail
09

Interactions

What happens when users interact with markers.

ActionResult
Click event markerOpens detail popover — title, location, time, source, and actions (Open, Save, Share)
Click stack markerOpens stack list — select any event from the group to see full detail
Click clusterZooms in to expand the cluster into individual markers
Hover analysis / newsShows a tooltip with the document or article title
Click map backgroundCloses the open popover and deselects the active marker
